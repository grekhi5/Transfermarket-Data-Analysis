# Transfermarket-Data-Analysis
Beyond the Pitch – Quantifying Success in Modern Football
This repository contains the Jupyter Notebook and analysis for A10: Beyond the Pitch – Quantifying Success in Modern Football, a data-driven exploration of modern football (soccer) performance using advanced metrics.
Project Overview
The goal of this project is to evaluate team performance beyond traditional win/loss outcomes. By leveraging statistical modeling, visualizations, and feature analysis, this notebook uncovers deeper insights into what drives success in modern football.
Key Objectives
Analyze team performance metrics using Python and pandas
Build interpretable visualizations to understand trends
Quantify relationships between variables such as goals, possession, and shot quality
Evaluate football teams based on holistic performance indicators
Technologies Used
Python (pandas, numpy, matplotlib, seaborn)
Jupyter Notebook for analysis and visualization

Repository Structure
├── A10-Beyond-the-Pitch-Quantifying-Success-in-Modern-Football.ipynb
└── README.md

Included Analysis
Data cleaning and preprocessing
Exploratory data analysis (EDA)
Correlation and feature importance exploration
Visualizations to interpret team strengths and weaknesses


How to Run the Notebook
Clone the repository:  git clone <your-repo-url>
Navigate into the project directory:  cd your-repo-name
Install required dependencies (optional):  pip install -r requirements.txt
Open the notebook: jupyter notebook A10-Beyond-the-Pitch-Quantifying-Success-in-Modern-Football.ipynb
Results & Insights

The analysis provides insights into:
